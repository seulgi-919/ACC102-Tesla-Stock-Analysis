# Tesla Stock Price Analysis (ACC102 Track 2)

A Python data analysis project for the XJTLU ACC102 Mini Assignment.

## Project Overview
This project analyzes Tesla (TSLA) stock price trends, volatility, and trading volume using Python.

## Problem Statement
The project explores the following question:

**How has Tesla's stock price changed over time, and what do trading volume and daily returns reveal about volatility and investment risk?**

## Target Users
- Beginner investors
- Finance students
- Users interested in basic stock market analysis

## Data Source
- Source: Yahoo Finance
- Ticker: TSLA
- Access method: yfinance
- Access date: 23 April 2026

## Tools
- Python
- pandas
- matplotlib
- yfinance

## Key Insights
1. TSLA experienced strong growth in earlier periods and high volatility afterward.
2. Volatility spikes may align with major market events such as earnings reports and product launches.
3. Daily returns show risk characteristics and frequent fluctuations.
