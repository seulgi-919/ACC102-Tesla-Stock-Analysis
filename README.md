# Tesla Financial & Stock Analysis (2023-2025)

## 1. Problem Definition
The goal of this project is to analyze the relationship between Tesla's quarterly financial performance (Revenue/Deliveries) and its stock price volatility. 
- **Target Audience:** Potential retail investors and automotive industry analysts.

## 2. Dataset Information
- **Sources:** Data was obtained from Yahoo Finance (stock prices) and Tesla Investor Relations (financial metrics).
- **Access Date:** April 2026.

## 3. Analysis Workflow
This project utilizes Python to:
1. **Data Acquisition:** Fetching Tesla stock data via `yfinance`.
2. **Data Cleaning:** Handling missing values and formatting dates.
3. **Exploratory Analysis:** Calculating moving averages and quarterly growth rates.
4. **Visualization:** Creating charts to show stock trends and revenue correlation.

## 4. How to Run
To run the notebook, install the following libraries:
`pip install pandas matplotlib seaborn yfinance`

## 5. Main Insights
Tesla's stock shows a strong correlation with quarterly delivery numbers.
The 50-day moving average suggests a bullish trend in Q4 2025.
